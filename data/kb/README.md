# ARIA AI Mentor — Knowledge Base

This directory holds the chunked knowledge base generated by `scripts/ingest_docs.py`.

## Files

| File | Description |
|---|---|
| chunks.jsonl | One JSON object per line — source file, heading, content, chunk_id, indexed_at |

## Rebuild

```bash
python3 scripts/ingest_docs.py /opt/aria-ai-mentor/repo
```

## Gitignore note

`chunks.jsonl` is gitignored — it is rebuilt from repo docs on demand, not committed. Only the ingestion script and this README are versioned.
